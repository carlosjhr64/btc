h1. BTC

BTC is a Ruby Library for Bitcoin.
This is version 0.0.0.
Currently, it only creates new bitcoin addresses, but
should be adding features over time.

h2. Synopsis

bc. # require 'rubygems'
# gem 'btc', '~> 0.0'
require 'btc'
bk = BTC::Key.new.generate_key
puts "New BTC Address:\t#{bk.address}"
puts "Private Key:\t\t#{bk.prvkey}"
puts bk.key.to_pem
